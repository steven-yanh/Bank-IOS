# Bank-IOS
This is a Bank project build for IOS environment that reveals many apple design principals: e.g

* `Delegate Protocol` 

* `Observer Pattern`

* `Programatic auto-layout`

* `MVVM design pattern`

Software/Hardware required for running the program:
`MacOs` `Xcode`
